# 📝 Smart Sticky Notes

A modern, responsive, and interactive Sticky Notes application built with **React.js** and **Tailwind CSS**. This project features a Pinterest-style masonry layout and a glassmorphism-inspired UI for a premium look.

## 🚀 Live Demo

https://sticky-notes-eosin.vercel.app/

## ✨ Features

  * **Add Notes**: Quickly create notes with a Title and Description.
  * **Real-time Validation**: Form submission is disabled until both fields are filled.
  * **Responsive Design**: Mobile-first approach using Tailwind CSS. The layout adjusts from a vertical stack on mobile to a multi-column view on desktop.
  * **Sticky Note UI**: Cards are styled with custom gradients, shadows, and a "paper pin" effect for a realistic feel.
  * **Delete Functionality**: Easily remove notes using the absolute-positioned delete button on each card.
  * **Interactive Hover Effects**: Cards feature smooth scaling and rotation transitions on hover.

## 🛠️ Tech Stack

  * **Framework**: [React.js](https://reactjs.org/) (Functional Components & Hooks)
  * **Styling**: [Tailwind CSS](https://tailwindcss.com/) (Utility-first CSS)
  * **Icons/Images**: Custom CSS gradients and external transparent PNGs.
  * **Build Tool**: Vite

## 📦 Installation & Setup

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/your-username/notes-project.git
    ```

2.  **Navigate to the project directory:**

    ```bash
    cd notes-project
    ```

3.  **Install dependencies:**

    ```bash
    npm install
    ```

4.  **Start the development server:**

    ```bash
    npm run dev
    ```

## 💡 Lessons Learned

During this project, I strengthened my understanding of:

  * Managing complex state arrays in React using `useState`.
  * Implementing responsive borders and layouts using Tailwind's breakpoint prefixes (`lg:`, `md:`).
  * The importance of using arrow functions in `onClick` handlers to prevent infinite render loops.
  * Using CSS `absolute` positioning for UI elements like pins and delete buttons.
